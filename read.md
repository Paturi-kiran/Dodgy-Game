# Dodgy Game 🎮

Dodgy Game is a simple 2D arcade-style game developed using the Godot Engine, where the player must avoid obstacles and survive as long as possible.

## 🚀 Features
- Player movement controls
- Collision detection system
- Basic obstacle mechanics

## 🛠️ Tech Stack
- Godot Engine
- GDScript

## ▶️ How to Run
1. Clone the repository
2. Open in Godot Engine
3. Run the main scene
