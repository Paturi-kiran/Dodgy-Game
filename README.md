# My-First-Project
## DodgeGame - 2D Godot Project
A fast-paced 2D game developed using the **Godot Engine** and **GDScript**. 

### **Key Features:**
*   **Dynamic Spawning**: Obstacles spawn at random intervals and positions.
*   **HUD System**: Integrated a Head-Up Display to manage game-over states and UI visibility.
*   **Signal Logic**: Implemented custom signals to connect the "Try Again" button to the scene restart logic.
*   **Version Control**: Managed the development lifecycle using Git and GitHub.

### **Technical Skills Demonstrated:**
*   Game Development (Godot/GDScript)
*   Logic Troubleshooting & Debugging
*   UI/UX Implementation
*   Professional Documentation
